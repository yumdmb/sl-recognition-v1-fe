Dynamic Learning Path
FR-009
The system shall allow users to take an initial evaluation to assess their current knowledge and goals.
FR-010
The system shall generate a personalized learning path based on the evaluation results, and learning performance.

---

Dashboard
FR-011
The system shall display the user's learning progress on the dashboard.
FR-012
The system shall allow users to navigate from the dashboard to another page.

---

Avatar Generation
FR-021
The system shall allow the User to capture a gesture image, preview the image and retake the capture.
FR-022
The system shall enable the User to enter the required gesture details and submit the gesture image.
FR-023
The system shall enable the Admin to manage avatar submissions (approve/reject) and update the submission status.

---

Learning
FR-024
The system shall allow the User to access and view a Tutorial and  update the tutorial status.
FR-025
The system shall allow the User to answer a Quiz, and upon completion, display the overall result and update the quiz status.
FR-026
The system shall allow the User to access the Downloadable Materials and download them.
FR-027
The system shall allow the Admin to select and view a list of all existing Tutorials, Quizzes, or Materials.
FR-028
The system shall allow the Admin to Add or Edit learning items, including managing individual questions and answers when editing Quizzes.
FR-029
The system shall allow the Admin to Delete selected learning items and save the changes to the database.

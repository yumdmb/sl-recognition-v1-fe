4.4.2 Software Requirements 
 
Software requirements include all the platforms, tools, and applications utilized in coding, testing, designing, and documenting the system. The key software used in this project are: 

| No. | Software | Roles |
|-----|----------|-------|
| 1 | Visual Studio Code (VS Code) | Used as the Integrated Development Environment (IDE) for writing, debugging, and managing source code. |
| 2 | MediaPipe | Used for developing and deploying real-time hand and gesture recognition models. |
| 3 | Blender | Used for 3D modeling and rendering, specifically for the avatar generation. |
| 4 | Next.js (Frontend) | Used to build the frontend of the website, enabling server-side rendering, optimized routing, and fast-loading interactive user interfaces. |
| 5 | Supabase (Backend) | Used as the backend-as-a-service (BaaS) providing PostgreSQL database, authentication, and real-time capabilities. |
| 6 | Draw.io | Used to visualize all the diagrams of the system |
| 7 | Figma | Used for the project mockup design |
| 8 | Canva | Used to prepare the slides for the project presentation |
| 9 | Google Docs | Used for the project documentation |
